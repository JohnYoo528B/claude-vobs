# 📥 收件箱

> **vault 的统一入口**。临时想法、外部资源、手机暂存——所有还没归类的都先丢进来。

## 四大角色

| 角色 | 场景 | 示例 |
|------|------|------|
| 🧠 临时想法 | 突然想到的点子、待整理笔记草稿 | 一个交易策略思路、一句感悟 |
| 📥 外部资源 | 别人发来的链接/论文/GitHub/文章 | Polymarket 数据集、一篇投资方法论 |
| 📱 手机暂存 | cc-connect / 微信发来的内容 | 路上想到的一句话、截图要存 |
| 🗺️ 画布草稿 | 头脑风暴、思维导图、还没成形的 `.canvas` | 策略图、项目架构草图 |

> ⚠️ **画布注意**：Obsidian 新建画布默认保存到根目录，不会自动进草稿箱。请手动选择保存到 `～草稿箱/`，或创建后拖进来。

## 外部资源摄入规则

从外部获取的资料存入时，AI 自动在 frontmatter 写入两个字段：

| 字段 | 说明 | 示例 |
|------|------|------|
| `source` | 原始链接 | `https://arxiv.org/abs/...` |
| `prerequisites` | 读之前需要先懂的概念 | `做市商机制, 订单簿` |

存入后标记 `status: unread`。启动时 AI 会按前置知识难度分级提醒你。

## 阅读状态流转

```
unread → active → archived
  ↓         ↓         ↓
还没读    正在学    已归档到对应主题文件夹
```

学完后说「学了 xxx」，AI 自动把 status 改为 `active`。消化完成、产生自己的判断后，升级到 `1主题研究/`。

## 子文件夹

| 文件夹 | 说明 |
|--------|------|
| `手机暂存/` | 手机端快速捕捉，按 session 分文件。说「存一下 xxx」触发。 |

## 整理规则

1. 说「**整理草稿箱**」触发清理：AI 逐文件分析 → 判定归类/合并/删除 → 呈现计划 → 确认后执行
2. 更新历史记录时也会清理一次
3. 每条内容归类到对应文件夹（`1主题研究/`、`2日志与笔记/`、`3工作与产出/`）
4. 归档完成后原始文件删除，或标记 `status: archived`
5. **不要让收件箱堆积超过一周**

## 快捷操作

- `Ctrl+N` 在收件箱新建笔记
- 内容写完后在 frontmatter 中标记 `status: draft`
- 定期说「整理草稿箱」逐条处理
