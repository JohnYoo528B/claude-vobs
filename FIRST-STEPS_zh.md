# 首次使用引导

> 你刚装好 Claude VObS。现在做什么？3 分钟，3 个操作。

---

## 1. 打个招呼（30 秒）

打开 Obsidian，找到 Claudian 对话框（侧边栏或命令面板），输入：

```
启动检查
```

AI 会回复系统当前状态——版本号、是否有未处理的操作、草稿箱积压情况。看到回复就说明系统跑通了。

---

## 2. 存第一条笔记（30 秒）

在对话框输入：

```
存 今天第一次用 VObS，感觉还不错
```

去 `～草稿箱/手机暂存/` 看看——应该多了一个新文件，里面有你刚说的话。

这就是"快速捕捉"——随时随地想到什么就存什么。手机上也能用（配好手机端后）。

---

## 3. 搜一下（30 秒）

在对话框输入：

```
找 CLAUDE.md
```

AI 会搜索 vault 里的相关文件。这就是 vault-search——搜你的笔记和对话记忆。

---

## 日常使用速查

| 你想做什么 | 怎么说 |
|-----------|--------|
| 快速记一个想法 | `存 <内容>` |
| 搜索笔记 | `找 <关键词>` |
| 把对话整理成笔记 | `整理成笔记` |
| 看看最近改了什么 | `改了什么` |
| 系统健康检查 | `启动检查` |
| 整理临时内容 | `整理草稿箱` |
| 系统回顾 | `回顾` |
| 发布到 GitHub | `发布 GitHub 新版本` |

---

## 不需要记触发词

系统设计就是让你说自然汉语。说「帮我看看最近改了啥」和说「改了什么」效果一样。说「把刚才聊的整理一下」和说「整理成笔记」也会路由到同一流程。

AI 自己判断你想做什么——你只管说，不用背命令。
